# Shikha Sarraf - Personal Portfolio

A clean, premium Light Mode personal portfolio website designed for desktop viewports. Built using HTML, CSS (Vanilla), and JavaScript.

##  Features
- **Facebook-style Profile Feed**: Interactive main feed featuring custom bio posts, stats, and Instagram highlights.
- **Dedicated About Page**: Showcases professional background and projects with borderless photos and clean offset boxes.
- **Projects Grid**: Academic & hackathon project cards with direct GitHub repository and live demo link buttons.
- **Blog Section**: Categorized into Tech articles, Personal reflections, and Education timeline blocks.
- **Contact Form**: Interactive form with built-in alert handling.

##  Technology Stack
- HTML5
- CSS3 (Vanilla design system)
- JavaScript (DOM manipulation)
